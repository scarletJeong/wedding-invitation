# 이미지 폴더

이 폴더에 청첩장에 사용할 이미지들을 넣어주세요.

## 필요한 이미지 파일

1. **메인 이미지**: `main.jpg` (또는 `main.png`)
   - 메인 섹션에 표시되는 큰 이미지
   - 권장 크기: 800x600px 이상

2. **갤러리 이미지**: 
   - `gallery1.jpg`
   - `gallery2.jpg`
   - `gallery3.jpg`
   - `gallery4.jpg`
   - 갤러리 섹션에 표시되는 사진들
   - 권장 크기: 600x600px 이상 (정사각형 권장)

3. **지도 이미지**: `map.jpg` (또는 `map.png`)
   - 오시는 길 섹션에 표시되는 지도 이미지
   - 네이버맵이나 카카오맵에서 스크린샷을 찍어서 사용하세요
   - 권장 크기: 600x400px 이상
   - 이미지 클릭 시 네이버맵으로 이동합니다

## 파일 형식
- JPG, PNG, WebP 모두 사용 가능
- 파일명은 소문자로 작성해주세요

## 사용 방법
1. 사진 파일들을 이 폴더(`public/images/`)에 복사하세요
2. 파일명을 위의 이름으로 변경하세요
3. 개발 서버를 재시작하면 자동으로 반영됩니다
